# 🧮 Simple Calculator Web App

A basic calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Simple and clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 How to Run
1. Download or clone the repository
2. Open index.html in your browser

## 🎯 Purpose
This project was created to practice basic JavaScript functions and DOM manipulation.
